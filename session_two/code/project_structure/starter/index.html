HelloWorld! - Your First Webpage

Welcome to the Web Page Design Course

This page provides the default folder structure to be used for all projects.

  • project_structure - the name of the folder.
    • img
      • img_one.jpeg
      • img_two.png
      • img_three.gif
    • css - contains all .css files necessary to change the look and feel of the web pages.
      • main.css
    • index.html - default web page file

Class Session Folders

To help keep track of class sessions, it's recommended that you create a folder for each class session.

For example, session_two would denote the all files related to session two of the course.
